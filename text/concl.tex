\conclusion

Мы протестировали два возможных расширения интерфейса регионов и предложили
мини язык для дальнейшего исследования одного из этих расширений. На основе
проделанного исследования можно сделать следующие выводы:

\begin{itemize}
\item Сроки жизни переменных при последовательной реализации задачи, как
  правило, пересекаются.
\item Для синтаксического представления этого пересечения необходимо в
  дополнение к блочной структуре создать структуру пересекающихся блоков,
  которая позволит более гибко управлять потреблением памяти.
\end{itemize}

В подтверждение первого тезиса автором этой работы было проведен анализ исходных
кодов нескольких программ на языке Java, подробно описанный в тексте работы. На
основе полученной информации был создан мини язык, который станет основой для
последующего изучения предложенной модели памяти.
